# Os 3 tipos de campanha que todo gestor de trafego precisa dominar no Meta Ads

Tres tipos fundamentais de campanha cobrem 90% das tecnicas de trafego pago:

## 1. Campanhas de Construcao de Audiencia
Cria audiencias qualificadas e as prepara para ofertas futuras. Cinco passos: definir metodos de atracao, planejar acoes de follow-up, selecionar objetivo certo, definir audiencia, criar anuncios persuasivos.

## 2. Campanhas de Geracao de Leads
Sete passos de implementacao: definir dados de contato, escolher destinos de campanha (Instagram, Facebook, WhatsApp), planejar atendimento pos-inscricao, criar mensagens de resposta, definir objetivos, configurar segmentacao, criar anuncios com CTAs claros.

Tres opcoes de objetivo: Engajamento (mensagens via WhatsApp, Direct, Messenger), Conversoes de landing page, Formularios instantaneos.

## 3. Campanhas de Conversao de Vendas
Objetivo primario: Maximizar vendas dentro do ROAS alvo. Cinco passos: definir oferta, escolher metodo de entrega, selecionar objetivo de campanha, definir segmentacao, criar anuncios eficazes.

## CONCLUSAO

Sucesso depende de dominar esses tres tipos de campanha e entender quando aplicar cada um.
