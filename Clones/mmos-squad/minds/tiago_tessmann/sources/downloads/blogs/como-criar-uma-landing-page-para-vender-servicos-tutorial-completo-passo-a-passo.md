# Como Criar uma Landing Page para Vender Serviços: Tutorial Completo

## O que é uma Landing Page

Uma landing page é uma página web focada em um objetivo único: conversão. Seja gerar leads, vender serviços ou atrair orçamentos, ela elimina distrações e guia visitantes para uma ação específica.

**Por que você precisa de uma:**
- Comunicação clara sobre um serviço específico
- Estrutura persuasiva para conversão
- Instalação de pixels de rastreamento (Meta, Google Ads)
- Melhor performance em campanhas pagas

## Requisitos Básicos Antes de Começar

### 1. Domínio
Nome na internet (ex: seuservico.com.br) – custa R$40-80/ano

### 2. Hospedagem
Serviços como HostGator oferecem planos a partir de R$9,39/mês com dois domínios grátis

### 3. Plataforma
Use WordPress + Elementor (gratuito, sem necessidade de programação)

## Passo a Passo Técnico

1. **Compre domínio e hospedagem** – preferencialmente juntos
2. **Instale WordPress** via painel de controle
3. **Instale plugin Elementor** (Plugins > Adicionar Novo)
4. **Crie página nova** com Elementor
5. **Adicione elementos principais:**
   - Imagem/vídeo de fundo
   - Headline poderosa
   - Subheadline explicativa
   - Botão de ação (CTA)
   - Depoimentos reais
   - Benefícios claros
   - Seção "Quem Somos"
   - FAQ

## Elementos Essenciais para Alta Conversão

- **Headline impactante** – conecte com a dor ou desejo do público
- **Subheadline clara** – explique rapidamente o que é seu serviço
- **Botão acima da dobra** – primeira ação visível sem rolar
- **Uma cor de destaque** – preto, branco + laranja ou verde para CTAs
- **Oferta imediata** – PDF de preços, análise gratuita ou avaliação
- **Prova social** – depoimentos e avaliações de clientes
- **Velocidade** – cada segundo de atraso reduz 6% da conversão

## Estratégias para Dobrar Conversão

- Crie versões para públicos e serviços diferentes
- Teste A/B em headlines, CTAs e imagens
- Instale Pixel do Facebook e Tag do Google
- Mantenha foco em uma ação única por página
- Faça follow-up rápido com leads

## FAQ

**Taxa média de conversão?** De 15% a 35%, dependendo do nicho

**CTA para WhatsApp ou formulário?** WhatsApp converte mais para serviços; formulário é melhor para nutrição de leads

**Como saber se funciona?** Instale pixels e acompanhe taxa de conversão, tempo na página e cliques
