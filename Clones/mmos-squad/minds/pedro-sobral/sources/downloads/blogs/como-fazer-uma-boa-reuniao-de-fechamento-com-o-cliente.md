# Como fazer uma boa reuniao de fechamento com o cliente

So existe uma profissao realmente: a de vendedor.

## Os Sete Estagios

1. Entender por que clientes contratam (seguranca, prova, resultados)
2. Requisitos essenciais da reuniao (pontualidade, preparacao, aparencia profissional)
3. Trabalho de preparacao (analise de anuncios, landing pages, Google Ads, velocidade, mobile, tags, perfis sociais)
4. Nivel de consciencia do cliente sobre trafego pago
5. Sete tipos de clientes (perdido, teorico, apertador de botao, experiente com agencia, frustrado, cetico, quase pronto)
6. Diretrizes de precificacao:
   - Iniciantes: Local R$300-1.000, E-commerce R$500-1.500, Infoprodutos R$1.000-3.000
   - Avancados: Local R$500-2.000, E-commerce R$1.500-5.000, Infoprodutos R$2.500-7.000
7. Estrategia de contratacao com atas notariais e confirmacao por email
