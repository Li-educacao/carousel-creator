# SecAppDev 2025 — Jim Manico

## Biography
Founder of Manicode Security. Java Champion. Author of "Iron-Clad Java." Fund-limited investor with Aviso Ventures, backing: SemGrep, EdgeScan, Nucleus Security, Defect Dojo, KSOC, Akto, MergeBase, Inspectiv, Levo.ai, Phoenix. Co-leads OWASP ASVS and Cheatsheet Series.

## Session 1: Using AI to Write Secure React.JS Code
- Deep-dive lecture (Monday June 2, 16:00-17:30)
- "Techniques that will aid AI code creation engine to produce higher quality and more secure code"

## Session 2: Secure Coding Workshop
- One-day hands-on (Friday June 6, 09:00-17:30)
- Covers: injection flaws, XSS, authentication weaknesses, insecure dependencies, OAuth2 basics, React security, file upload safety, AI-assisted code generation
- Target: "Developers seeking to strengthen their security mindset and threat-resistant applications"
