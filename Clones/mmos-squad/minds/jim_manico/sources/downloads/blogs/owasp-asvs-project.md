# OWASP ASVS (Application Security Verification Standard)

## Purpose
Provides "a basis for testing web application technical security controls" and "a list of requirements for secure development."

## Leadership
- Daniel Cuthbert, Jim Manico, Elar Lang, Josh Grossman (project leaders)
- Jim Manico: Lead Author Version 2 (2014), continuing project leader
- Manicode is a Maintaining Supporter

## Current Version: 5.0.0 (released May 30, 2025, Global AppSec Barcelona)

## Requirement Format
Structure: chapter.section.requirement (e.g., 1.11.3)
Example: "Verify that the application protects against OS command injection and that operating system calls use parameterized OS queries or use contextual command line output encoding."

## Adoption
- CREST OWASP Verification Standard (OVS)
- App Defense Alliance CASA (Cloud Application Security Assessment)
- Used by Booz Allen Hamilton, CGI Federal, Denim Group, NVISO, and many others

## Available Formats: PDF, Word, CSV, JSON
